# nrb_forex_tracker
A MERN application to track Foreign Currency Exchange Rates against Nepalese currency using API provided by Nepal Rastra Bank. 

## Features
### Exchange Rate Reminder
Logged in users can set reminders for their desired exchange rate of Nepalese currency against a Foreign Currency offered in exchange by the Nepal Rastra Bank. When the desired exchange appears in NRB's API, we notify users via email.
